# Revenue Analytics — Full-Page Stripe App Example

This example demonstrates how to build a **full-page Stripe App** using `FullPageView`.

## Running the example

1. Install dependencies:

   ```sh
   pnpm install
   ```

2. Start the app in development mode:

   ```sh
   stripe apps start
   ```

3. Open the Stripe Dashboard and navigate to the app.
